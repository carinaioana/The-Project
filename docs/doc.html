Unemployment Explorer

Autori: Sirbu Carina Ioana si Astefanoae Ioana

Rezumat

Această documentație descrie un proiect de statistici privind șomajul în România, realizat cu ajutorul unui bar chart, un pie chart și o hartă cartografică, care filtrează informațiile în funcție de numărul total, rate, gen, și tipul de plată. Acest proiect a avut ca scop analiza tendințelor în ceea ce privește șomajul din România, precum și identificarea diferențelor în funcție de gen și tipul de plată.

Metodologie

Tehnologii utilizate: HTML, CSS, JavaScript

Structura fișierelor:

- api: documentul JavaScript care realizează analiza datelor și construiește graficele și harta

- style.css: fișierul CSS pentru stilizarea paginii

- static: documentul in care se gasesc datele despre somajul populatiei pe judete

- ro.csv: fișierul CSV care conține datele despre șomaj în România

- views: documentul cu toate paginile html

- home.html: fișierul principal HTML, conține structura paginii și legăturile cu fișierele CSS și JS

Structura HTML:

- head: conține metodatele paginii și legăturile cu fișierele CSS și JS

- body: conține elementele vizibile ale paginii, cum ar fi antetul, conținutul principal și subsolul
